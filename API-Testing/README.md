API Testing

This folder contains the API Testing work performed for the Dummy JSON Application.

The project includes API testing using Postman and API automation using REST Assured. APIs related to Authentication, Products, Cart, Wishlist, Orders, Coupons, User Profile, and Address Management were tested and validated.

Using Postman, I performed request creation, response validation, request chaining, environment management, mock API testing, and generated execution reports using Newman.

Using REST Assured, I automated the APIs using Java, Maven, and TestNG. The framework includes GET, POST, PUT, and DELETE request handling along with response validation, assertions, and reporting.
