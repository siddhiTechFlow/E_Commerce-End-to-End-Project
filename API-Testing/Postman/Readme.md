# API Testing Using Postman

## Project Overview

This project contains API testing performed using Postman for an E-Commerce application. The purpose of this project was to validate different APIs related to authentication, products, cart management, wishlist management, orders, coupons, profile management, and address management.

Both real APIs (DummyJSON) and Postman Mock APIs were used to simulate real-world API testing scenarios.

---

## Tools Used

* Postman
* Newman
* Postman Mock Server
* Git & GitHub

---

## APIs Covered

### Authentication

* User Registration
* User Login
* Send Reset Password Email
* Get Current User Profile

### Products

* List All Products
* Get Product By ID
* List Categories

### Cart

* Add Item To Cart
* Get Current User Cart
* Update Cart Quantity
* Remove Item From Cart

### Wishlist

* Add Product To Wishlist
* Get Wishlist

### Orders

* Place Order
* Get All Orders
* Get Order Details
* Cancel Order

### Coupons

* Validate Coupon

### User Management

* Update User Profile
* Add Delivery Address
